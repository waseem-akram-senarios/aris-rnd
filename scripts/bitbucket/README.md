# Bitbucket Setup Scripts

One-time scripts for configuring Bitbucket access and repository setup.

| File | Purpose |
|------|---------|
| `BITBUCKET_SETUP_GUIDE.txt` | Setup instructions |
| `check_bitbucket_access.py` | Verify SSH/HTTPS access |
| `complete_bitbucket_setup.py` | Full setup automation |
| `setup_bitbucket_git.py` | Git remote configuration |
| `setup_bitbucket_https.py` | HTTPS credential setup |
